# 算法岗面试
**该项目主要分享在准备算法岗面试过程中学习的应具备的八股知识和手撕代码题目，以帮助更快速地通过算法岗实习/正式职面试。**
- 
## 目录
- [机器学习相关](https://github.com/wenshou157/intern-for-algorithm/blob/main/ml.ipynb)
- [深度学习相关](https://github.com/wenshou157/intern-for-algorithm/blob/main/dl.ipynb)

该项目主要收集参考不同大语言模型（如ChatGPT、deepseek、Groks、豆包等）给出的学习路线、小红书学习路线帖子和面经等内容，试图完成从0找到算法岗的全链路学习。其中各章节内容部分经过大语言模型生成，再经过梳理修改完善而成，希望能够提供一些帮助。
**剩余内容正在持续更新中**
